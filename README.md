# sequencing

順番決めができます。

インプットボックスに数値を入力すると、文字列が入力できるインプットボックスが現れるので、インプットボックスに順番決めしたい名前などを入力してシャッフルボタンをクリック(タップ)すると、シャッフルされた値が出力されます。
